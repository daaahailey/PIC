## 포도마켓은,  
- 🦁멋쟁이사자처럼 프론트엔드스쿨 1기에서 팀 프로젝트로 진행한 포도마켓입니다.
- 판매할 상품을 등록하여 홍보, 판매할 수 있습니다.
- 또한 판매가 아니더라도 사진과 글을 작성하여 일상을 공유할 수 있는 SNS입니다.
- 다른 사용자를 팔로우하여 사람들의 소식을 확인할 수 있습니다.
- 게시글에 좋아요와 댓글을 남길 수 있습니다.
- 🍇알알이 이웃들이 모여 이룬 풍성한 한 송이, 포도마켓입니다.
